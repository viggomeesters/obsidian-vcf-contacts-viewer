# Changelog

## 0.1.0

- Initial read-only `.vcf` viewer.
- Added vCard 3.0 and 4.0 parsing for common contact fields.
- Added multi-contact file support.
- Added contact list, detail view, raw source view, search, warnings, and refresh.
- Added import-preflight warnings for unsupported fields, malformed cards, version mismatches, folded-line issues, encoded fields, and PHOTO handling.
