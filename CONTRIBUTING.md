# Contributing

VCF Contacts Viewer is intentionally narrow: it opens `.vcf` files as read-only inspection views.

Useful contributions:

- Better synthetic fixtures for vCard 3.0 or 4.0 edge cases.
- Parser warnings that preserve raw fallback behavior.
- Accessibility and responsive layout improvements.
- Documentation improvements around privacy and import-preflight review.

Out of scope for v0.1:

- Contact management.
- Contact note creation or updates.
- Export or write-back to `.vcf`.
- Click-to-call, mailto, FaceTime, Contacts.app, or external app launch actions.
- Remote image loading or avatar import.
- Clipboard features.
